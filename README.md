# 튜토리얼 따라하기

starter 브랜치로 checkout하여 튜토리얼을 바로 시작하실 수 있습니다.

```
git clone https://github.com/alexKwonIsAwesome/simplenote-react
cd simplenote-react
git checkout starter
yarn start
```
